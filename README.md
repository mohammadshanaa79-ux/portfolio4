🛍️ Luxe Store - Luxury E-commerce Platform
A modern, premium e-commerce web application built with React, featuring 120+ luxury products across 8 categories with a sophisticated user interface and smooth shopping experience.

✨ Key Features
Product Catalog: 120+ premium products from luxury brands
8 Categories: Electronics, Fashion, Home, Beauty, Sports, Books, Jewelry, Watches
Shopping Cart: Add, remove, and manage products
Wishlist: Save favorite items
Search & Filter: Advanced product filtering and sorting
Quick View: Preview products without leaving the page
Dark/Light Mode: Toggle between themes
Responsive Design: Works on all devices
Smooth Animations: Modern UI with Framer Motion
🛠️ Tech Stack
React 18 - UI library
React Router - Navigation
Zustand - State management
Tailwind CSS - Styling
Framer Motion - Animations
Heroicons - Icons
📁 Project Structure
src/
├── components/    # Reusable UI components
├── pages/         # Page components
├── hooks/         # Custom React hooks
├── utils/         # Helper functions
├── services/      # API services
├── constants/     # App constants
├── store/         # State management
└── data/          # Product data
🚀 Getting Started
npm install
npm start
Open http://localhost:3000 to view the app.

📦 Main Pages
Home - Featured products and categories
Shop - All products with filters

Cart - Shopping cart management
Checkout - Multi-step checkout process
Dashboard - User orders and wishlist
