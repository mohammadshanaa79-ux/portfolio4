# 🍔 URBAN BITES Restaurant

**Fresh Food, Urban Vibes** ✨

Modern restaurant website built with Next.js 14, featuring online ordering, admin dashboard, and real-time updates.

---

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Setup database
npx prisma db push
node prisma/seed.js

# Run development server
npm run dev
```

Visit: http://localhost:3000
