# 🤝 المساهمة في المشروع

نرحب بمساهماتك في تطوير مشروع URBAN BITES Restaurant!

## 📋 كيفية المساهمة

### 1. Fork المشروع
اضغط على زر "Fork" في أعلى الصفحة

### 2. استنسخ المشروع
```bash
git clone https://github.com/YOUR_USERNAME/shik-shak-restaurant.git
cd shik-shak-restaurant
```

### 3. أنشئ فرع جديد
```bash
git checkout -b feature/amazing-feature
```

### 4. قم بالتعديلات
- اكتب كود نظيف ومنظم
- اتبع معايير TypeScript
- أضف تعليقات واضحة

### 5. Commit التغييرات
```bash
git add .
git commit -m "إضافة ميزة رائعة"
```

### 6. Push للفرع
```bash
git push origin feature/amazing-feature
```

### 7. افتح Pull Request
افتح Pull Request من فرعك إلى الفرع الرئيسي

## 📝 معايير الكود

- استخدم TypeScript
- اتبع معايير Tailwind CSS
- اكتب كود RTL-friendly
- اختبر التغييرات قبل الـ Push

## 🐛 الإبلاغ عن مشاكل

إذا وجدت مشكلة، افتح Issue جديد مع:
- وصف واضح للمشكلة
- خطوات إعادة إنتاج المشكلة
- لقطات شاشة (إن أمكن)

## 💡 اقتراح ميزات

نرحب باقتراحاتك! افتح Issue مع:
- وصف الميزة المقترحة
- الفائدة المتوقعة
- أمثلة (إن وجدت)

شكراً لمساهمتك! 🇵🇸❤️
